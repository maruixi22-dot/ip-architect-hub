# IP Architect Hub

> 自媒体 IP 全流程孵化系统

## 📋 项目简介

IP Architect Hub 是一个完整的自媒体 IP 孵化平台，帮助用户从 0 到 1 打造和管理多个 IP 账号。

### 核心功能

1. **IP 调研** - 通过语音/文字问卷完成 IP 人物画像
2. **运营规划** - AI 自动生成 IP 运营规划书
3. **内容采集** - 自动采集小红书/抖音热门内容
4. **智能推送** - 定时推送优质内容到用户
5. **AI 改写** - 一键改写符合 IP 调性的内容
6. **多平台发布** - 自动发布到多个平台
7. **数据分析** - 实时监控 IP 数据表现
8. **多 IP 管理** - 统一管理多个 IP 项目

```
┌─────────────────────────────────────────────────────────────┐
│                    用户界面层                                │
│              Next.js 14 + shadcn/ui                         │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│                    应用层 (API Routes)                      │
│                    Next.js API Routes                        │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│                    服务层                                    │
├─────────────────────────────────────────────────────────────┤
│  AI 服务    │  GLM-5 API (智谱 AI)                          │
│  语音服务  │  Whisper API (OpenAI)                         │
│  爬虫服务  │  Puppeteer / Cheerio                          │

│  MongoDB     │  采集内容、日志、非结构化数据                │
│  Redis       │  缓存、任务队列、会话                         │
│  OSS/S3      │  文件存储（音频、图片、视频）                │
```

## 🚀 快速开始

### 环境要求

- Node.js 18+
- PostgreSQL 14+
- Redis 6+

### 安装

```bash
# 克隆项目
git clone https://github.com/yourusername/ip-architect-hub.git
cd ip-architect-hub

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local

# 初始化数据库
npm run db:setup

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000
